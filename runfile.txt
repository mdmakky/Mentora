# MENTORA - Easy Way to Run Frontend and Backend

## EASIEST METHOD: Use Two Terminals

### Terminal 1 (Backend):
cd /home/muse-matrix/Desktop/Mentora/backend
python manage.py makemigrations
python manage.py migrate
python manage.py runserver

### Terminal 2 (Frontend):
cd /home/muse-matrix/Desktop/Mentora/frontend
npm start

## QUICK REFERENCE:
- Backend runs on: http://localhost:8000
- Frontend runs on: http://localhost:3000
- Open frontend in browser: http://localhost:3000

## TO STOP SERVERS:
- Press Ctrl+C in each terminal window
